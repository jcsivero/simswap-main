Léame de los Servicios de autenticación Internet

Antes de instalar este producto, lea este documento en su totalidad y lea el Manual del administrador del Servidor de autenticación de Internet de Microsoft (IAS). El documento contiene información crítica para asegurar la correcta instalación y uso del producto.

Importante   Este archivo léame contiene información importante de última hora acerca de este producto.

Importante para los clientes de la beta 3:

  • La información de configuración, almacenada en Adminui.mdb, se quedará como está. Esta versión utiliza un nuevo archivo _Adminui.mdb. Para garantizar la seguridad de los datos actuales, copie Adminui.mdb a otra ubicación.
  • Antes de instalar esta versión de IAS en un equipo que ejecute la versión Beta 3, debe detener los Servicios de autenticación Internet y desinstalarlos. (Si tiene que volver a instalar la versión final de IAS en algún momento, también tendrá que detener IAS y desinstalarlo primero.)

Problemas conocidos de Servicios de autenticación Internet

  • En el programa de instalación, Servicio de autenticación Internet se muestra como un subcomponente de Servicios de conexión de Internet para Servicios de acceso remoto bajo Windows NT4.0 Option Pack. Tendrá que activar la casilla de verificación de Servicio de autenticación Internet para instalar este servicio.
  • Si es la primera vez que instala IAS, no se iniciará IAS hasta que lo configure mediante la herramienta de administración Propiedades del Servicio de autenticación Internet.

    Para configurar el Servicio de autenticación Internet:
    1. Haga clic en Inicio, Programas, Windows NT 4.0 Options Pack, Microsoft Internet Information Server y Administrador de servicios de Internet. Aparecerá Microsoft Management Console (MMC).
    2. En el panel izquierdo de MMC, haga doble clic en Internet Information Server y, después, haga doble clic en el nombre de la máquina local. Bajo esta máquina se muestra el servicio RADIUS.
    3. Haga clic con el botón secundario del mouse (ratón) en RADIUS y, después, haga clic en Propiedades.
    4. Configure el servidor y haga clic en Aceptar o en Aplicar para guardar los cambios.
  • Algunos servidores de acceso a redes no aceptan contraseñas mayores de 16 bytes cuando utilizan el Protocolo de autenticación de contraseñas (PAP) del cliente. Microsoft no puede garantizar que las contraseñas mayores de 16 se autentiquen correctamente con todo el equipamiento de acceso.
  • La administración remota de IAS no funcionará si las rutas de instalación de IAS son diferentes en el servidor local y en el servidor remoto. Si va a utilizar la administración remota en su red, compruebe que todos los servidores de IAS tienen las mismas rutas de instalación para IAS.
  • IAS aceptará el Protocolo de autenticación de contraseña (PAP) o Microsoft CHAP (similar a Challenge Handshake Authentication Protocol (CHAP, Protocolo de autenticación de Desafío)). En una versión futura se incluirá también compatibilidad con CHAP.
  • Si desinstala Microsoft(r) Access 97 o Visual Basic(r) de un equipo que ejecuta Servicios de autenticación Internet, debe volver a instalarlo una vez completada la desinstalación.
  • Para desinstalar IAS:
    1. Haga clic en Inicio, Programas, Windows NT 4.0 Options Pack, Microsoft Internet Information Server, Administrador de servicios Internet. Aparecerá Microsoft Management Console (MMC).
    2. En el panel izquierdo de MMC, haga doble clic en IIS y, después, haga doble clic en el nombre de la máquina local. Debajo de esa máquina aparecerá el servicio RADIUS.
    3. Haga clic con el botón secundario del mouse (ratón) en RADIUS y, después, haga clic en Detener.
    4. Inicie el programa de instalación de Windows NT 4.0 Option Pack y, después, haga clic en Siguiente y en Agregar o quitar.
    5. En la lista Componentes, haga clic en Servicios de acceso remoto y, después, haga clic en Mostrar subcomponentes.
    6. Seleccione Servicios de autenticación Internet y haga clic en la casilla de verificación para desactivarla.
    7. Haga clic en Aceptar y en Siguiente y, después, siga las instrucciones de la pantalla.
    8. Cuando el programa de instalación haya terminado de quitar IAS, reinicie el equipo.
  • Para volver a instalar IAS:
    1. Inicie el programa de instalación de Windows NT 4.0 Option Pack y, después, haga clic en Siguiente y en Agregar o quitar.
    2. En la lista Componentes, haga clic en Servicios de acceso remoto y, después, haga clic en Mostrar subcomponentes.
    3. Haga clic en la casilla de verificación Servicios de autenticación Internet.
    4. Haga clic en Aceptar y en Siguiente y, después, siga las instrucciones de la pantalla.
    5. Cuando el programa de instalación haya terminado de instalar IAS, reinicie el equipo.

Para obtener más información
