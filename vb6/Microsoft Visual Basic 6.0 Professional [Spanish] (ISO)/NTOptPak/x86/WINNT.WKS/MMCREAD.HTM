Bienvenidos a MMC versión 5.0

Bienvenidos a la versión beta 3 de MMC de Internet Information Server. MMC integra en una única consola todas las herramientas necesarias para realizar determinadas tareas administrativas. Para crear una nueva consola, un administrador de sistema agrega a MMC herramientas administrativas como complementos. Estos complementos pueden ser las herramientas administrativas estándar para Windows, que se están convirtiendo en complementos. También pueden ser herramientas administrativas configuradas como complementos por proveedores independientes de software. Al agregar componentes a una consola, un administrador puede ensamblar todas las herramientas y la información que necesita para evaluar y configurar un componente del sistema. Es posible agregar a una consola incluso vínculos con herramientas basadas en Web, páginas Web que proporcionen información o actualizaciones a componentes del sistema.

Una vez terminada una consola, el administrador puede guardarla en un archivo .msc (Management Saved Console). De manera predeterminada está disponible en el grupo Mis herramientas de administración, que se encuentra en Programas, bajo el menú Inicio. El administrador puede entregar la consola a otras personas en las que haya delegado las tareas administrativas. Es posible enviar la consola por correo electrónico, ponerla en una carpeta compartida de la red o asignale directivas del sistema. Cuando se abre la consola, aparece con la misma configuración exacta con que se guardó.

Ejecutar MMC

Importante    Esta versión de MMC 5.0 IIS beta 3 está destinada a fines de prueba únicamente. Microsoft recomienda que ejecute esta versión en un entorno de prueba y no para tareas críticas.

Para abrir las consolas administrativas estándar de Microsoft, haga clic en el menú Inicio, Programas, Herramientas administrativas y, después, haga clic en la herramienta que desee abrir. También puede hacer clic en un archivo .msc desde el Explorador de Windows para abrirlo.

Para crear una nueva consola, en el menú Inicio, haga clic en Ejecutar, escriba mmc y haga clic en Aceptar.

Documentación del producto

Toda la documentación del producto está disponible en pantalla. Cuando MMC está en ejecución, puede tener acceso a los temas de Ayuda de MMC si hace clic en Ayuda de MMC en el menú Ayuda.

Para ver ayuda acerca de un complemento de una consola de MMC, haga clic en el complemento y, en el menú Ayuda, haga clic en Ayuda de complemento.

Si Ayuda de complemento no está disponible, significa que no hay Ayuda para el complemento seleccionado.

Obtener soporte técnico y enviar comentarios

Para ver noticias acerca de MMC, en el menú Ayuda de MMC, seleccione Microsoft en el Web y haga clic en Noticias de productos. Esto le llevará a la página Web de MMC en el sitio Web de Microsoft. En dicha página se ofrece una introducción a MMC, así como las preguntas más frecuentes e información acerca de cómo escribir complementos.

Para enviar comentarios acerca de MMC, seleccione Microsoft en el Web y haga clic en Enviar comentarios. Aparecerá una página Web que explica cómo enviar comentarios acerca del producto.

Otros problemas y limitaciones conocidos

La siguiente lista contiene algunos problemas y limitaciones que se sabe que existen en esta versión beta 3:

La Ayuda puede no estar totalmente implementada
El modo de sólo lectura no está disponible

    La Ayuda puede no estar totalmente implementada

  • En esta versión no está disponible la Ayuda contextual para los cuadros de diálogo de MMC.
  • La Ayuda sólo está implementada para algunas de las consolas administrativas estándar incluidas con esta versión de MMC.

    El modo de sólo lectura no está disponible

  • El modo de sólo lectura no está disponible en esta versión. Las futuras versiones de MMC proporcionarán un modo de sólo lectura para asegurar que los usuarios no modifiquen los archivos de consola suministrados con las aplicaciones.
