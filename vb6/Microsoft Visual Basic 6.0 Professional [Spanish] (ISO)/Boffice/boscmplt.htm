BackOffice Server Complete Installation

This BackOffice Server business solution is designed for small networks containing only a few servers. It installs the most commonly used BackOffice Server components.

Because of their specialized use, Systems Management Server (SMS) and SNA Server are optional.

The total amount of disk space required for the complete installation is about 520 MB to 660 MB, depending on whether you install SMS and SNA Server. Additional disk space may be required for SQL Server database storage, Proxy Server cache, and Exchange Server mail storage.

Before Setup begins installing components, you will be able to customize the installation to meet your unique requirements. You can choose to add or remove components from the installation, or to change default settings (such as folder locations).

For more information about BackOffice Server business solutions, see BackOffice Live, the Microsoft BackOffice site on the World Wide Web.

The complete installation provides the following BackOffice Server components:

  • Exchange Server 5.5
  • Intranet Starter Site 1.0
  • Microsoft Internet Explorer 4.01
  • Proxy Server 2.0
  • SNA Server 4.0 (optional)
  • Seagate Crystal Info 5.0
  • SQL Server 6.5
  • Systems Management Server 1.2 (optional)
  • Windows NT 4.0 Option Pack
