Confirming Your Installation Choices

Use this dialog box to review how BackOffice Server Setup will configure your server.

Items with a plus sign (+) next to them have more detail. Click the plus sign to see additional configuration about the item.

If the configuration is acceptable, click Next. To change an item that you had selected in an earlier dialog box, click Back. For complete control over how Setup installs BackOffice Server on your computer, click Customize My Solution.
