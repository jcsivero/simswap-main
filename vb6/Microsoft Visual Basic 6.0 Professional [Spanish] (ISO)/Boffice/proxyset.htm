Proxy Server Settings

This dialog presents default settings for your Proxy Server configuration. To enable an item, select the check box next to the item. To change the default value assigned to an item, select the item and then click Change.

Caching configuration

If you choose to enable file caching (the default), you must specify the drive on which cached data will be stored and the amount of disk space to be used for caching. Default: 100 MB on drive C

Winsock proxy server access mode

You must specify whether Winsock clients will access Proxy Server using the server IP address or using the server name. Default: IP address

Automatic Web browser configuration

If you set Proxy Server to configure client Web browsers automatically (the default), you must specify the server name, port, and configuration URL of the configuration script. Default: determined dynamically during installation

Winsock and Web proxy access security

You must specify whether to control access to Winsock and Web Proxy services. If enabled, the services attempt to validate connections from clients. Default: control access
