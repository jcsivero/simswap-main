Proxy Server Local Address Table Settings

The Local Address Table (LAT) consists of a series of IP address pairs that define your internal network address space. Each address pair defines either a range of IP addresses or a single IP address.

BackOffice Server Setup automatically constructs the table using the list of IP address pairs found in the Windows NT Server routing table. It is important that you review the generated list of IP addresses. In some cases, the generated addresses might not completely define your internal network. The list may omit a subnet of your internal network or may include addresses external to your internal network. You can add or remove specific address pairs until all the internal network addresses are defined. Be sure to remove any IP address pairs that define external (Internet) addresses.
