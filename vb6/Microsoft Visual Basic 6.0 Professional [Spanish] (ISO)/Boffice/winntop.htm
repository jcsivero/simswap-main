Microsoft Windows NT Server 4.0 Option Pack

The Option Pack enhances Windows NT Server with a set of Internet-enabled server applications that extend your mission-critical applications to your corporate intranet or to the Internet. The Option Pack consists of the following components:

  • Microsoft Internet Information Server 4.0
  • Microsoft Management Console 1.0
  • Microsoft Index Server 2.0
  • Microsoft Certificate Server 1.0
  • Microsoft Data Access Components 1.5
  • Microsoft Site Server Express 2.0
  • Microsoft Message Queue 1.0
  • Microsoft Remote Access Services 1.0
  • Microsoft Script Debugger 1.0

For more information, see Microsoft BackOffice Server Getting Started.
