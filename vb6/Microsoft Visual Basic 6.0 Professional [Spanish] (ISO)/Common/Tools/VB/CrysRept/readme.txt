Crystal Reports for Visual Basic 6.0 Readme


This readme includes the limitation of Crystal Reports for Visual Basic 6.0.

1. This Crystal Reports is same version as one in Visual Basic 5.0.

2. Some "Visual Basic 5.0" word remains in Readme.hlp and Crw.hlp. But you can read it as "Visual Basic 6.0" word.

3. You can execute Crystal Reports by clicking "Crw.exe" from the installed folder directly or by selecting "Crystal Reports" from Add-In menu. This menu is added by Crystal setup. Crystal Reports isn't registried to Windows "Start" menu.
